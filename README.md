#OBD Features
* Read fault codes
* Clear fault codes
* Read/record live data
* Read freeze frame data
* Read vehicle info data

#Additional features
* Save recorded data
* Load recorded data (for analysis)
* CSV export
* Data charts
* Dashboard
* Head up display
* Custom PID data
* Day/Night view
